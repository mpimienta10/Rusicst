Extensiones de tiempo

Las extensiones de tiempo permiten al usuario administrador otorgar un nuevo plazo para completar un Reporte, Plan de Mejoramiento o Tablero PAT. El plazo se asigna a un usuario específico sobre un reporte específico, por lo que deberá seleccionar estos dos.

Seleccione un Tipo de Extensión de la lista: Encuesta (Reporte), Plan de Mejoramiento o Tablero PAT.

Para seleccionar el usuario simplemente vaya escribiendo algunas de las letras contiguas del nombre de este o del nombre de usuario con el que accede al sistema.

De igual manera proceda con la selección del Reporte, Plan de Mejoramiento o Tablero PAT, escribiendo parte del nombre de este para que el sistema le muestre el listado de reportes que cumplen con el término buscado.
